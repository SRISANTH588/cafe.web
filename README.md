# Ayyan Cafe - Complete Package

## 📦 All Files in One Folder

This folder contains everything you need for your cafe system.

## 📁 Files Included:

### Main Websites:
- **cafe.html** - Main cafe ordering website
- **payment.html** - Standalone payment website

### Scripts & Styles:
- **cafe-script.js** - All cafe functionality
- **cafe-styles.css** - All styling

### Backend:
- **api-server.js** - API server (for different domains)
- **server.js** - Paytm payment server

### Configuration:
- **package.json** - Dependencies
- **vercel.json** - Vercel deployment config
- **cafe-config.js** - Cafe settings

## 🚀 How to Use:

### Option 1: Local Testing
```bash
npm install
node api-server.js
```
Open: http://localhost:3000/cafe.html

### Option 2: Deploy to Vercel
1. Upload this entire folder to GitHub
2. Connect to Vercel
3. Deploy!

### Option 3: Deploy to GitHub Pages
1. Upload to GitHub repository
2. Enable GitHub Pages
3. Access via: https://username.github.io/repo-name/cafe.html

## 📝 Change Cafe Name:

Open cafe.html → Click ☰ → Cafe Name & Branding → Change → Save
